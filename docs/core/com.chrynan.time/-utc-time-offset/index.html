UtcTimeOffset

(JVM, COMMON) @ExperimentalTime enum class UtcTimeOffset

An enum class representing a group of common UTC TimeOffsets. The offset value returns the actual TimeOffset. Note that this may not be a comprehensive list and for offsets that are not provided by this enum class, use the TimeOffset class directly.

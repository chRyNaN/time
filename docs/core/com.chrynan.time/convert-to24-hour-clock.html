convertTo24HourClock

(JVM, COMMON) fun HourInDay.convertTo24HourClock(): HourIn24HourClock

Converts this HourInDay instance to a HourIn24HourClock. If this HourInDay is already a HourIn24HourClock, then this is returned but scoped appropriately. Otherwise a conversion is performed from HourIn12HourClock to HourIn24HourClock.

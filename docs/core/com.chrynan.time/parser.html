(JVM, COMMON) @ExperimentalTime fun TimeProvider.parser(pattern: String): MomentParser

Retrieves a MomentParser for the provided pattern for converting Strings into Moments. This is a convenience function that converts the provided pattern to a MomentFormatPattern, and then calls the TimeProvider.parser function.

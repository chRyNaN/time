toUtc

(JVM, COMMON) fun toUtc(): Moment

Retrieves a new Moment representing the same time as this Moment but in the UTC/GMT region. This is similar to subtracting the offsetFromUtc from this Moment and updating the timeZoneRegionId to be DefaultTimeZoneRegionId.UTC.

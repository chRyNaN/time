durationSinceTheEpoch

(JVM, COMMON) val durationSinceTheEpoch: Duration

The amount of time from the unix epoch (00:00:00 UTC on 1 January 1970). The value is a Duration representing the difference between the current moment and the unix epoch moment. This value can be negative if it is before the epoch.

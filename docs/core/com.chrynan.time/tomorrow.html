tomorrow

(JVM, COMMON) @ExperimentalTime fun TimeProvider.tomorrow(regionId: TimeZoneRegionId = timeZoneOffsetProvider.defaultTimeZoneRegionId): Moment

Retrieves the Moment representing the start of the day after today. This is a convenience function for calling TimeProvider.now followed by Moment.startOfDay and then adding a one day Duration.

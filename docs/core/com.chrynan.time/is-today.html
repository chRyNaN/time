isToday

(JVM, COMMON) @ExperimentalTime fun TimeProvider.isToday(moment: Moment, regionId: TimeZoneRegionId = timeZoneOffsetProvider.defaultTimeZoneRegionId): Boolean

Determines whether the provided moment is the same date as today. This is a convenience function for calling TimeProvider.now and then calling Moment.isSameDateAs on that and the provided value to determine if they are the same date.

moment

(JVM, COMMON) fun moment(durationSinceEpoch: Duration, regionId: TimeZoneRegionId): Moment

Retrieves a Moment for the provided durationSinceEpoch and regionId. If no regionId is provided the default obtained by the TimeZoneOffsetProvider.defaultTimeZoneRegionId is used. The resulting Moment is relative to the provided regionId. This is the same as obtaining the Moment at UTC/GMT for the durationSinceEpoch and then offsetting it with the regionId.

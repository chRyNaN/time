now

(JVM, COMMON) fun now(regionId: TimeZoneRegionId): Moment

Retrieves a Moment representing the current moment in time for the provided regionId. If no regionId is provided then the default obtained by the TimeZoneOffsetProvider.defaultTimeZoneRegionId is used. The resulting Moment is relative to the provided regionId.

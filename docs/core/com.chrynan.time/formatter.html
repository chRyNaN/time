(JVM, COMMON) @ExperimentalTime fun TimeProvider.formatter(pattern: String): MomentFormatter

Retrieves a MomentFormatter for the provided pattern for converting Moments into Strings. This is a convenience function that converts the provided pattern to a MomentFormatPattern, and then calls the TimeProvider.formatter function.

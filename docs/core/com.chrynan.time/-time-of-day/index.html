TimeOfDay

(JVM, COMMON) @ExperimentalTime data class TimeOfDay

A class that represents the exact time within a Calendar date. This represents a moment of time and not an elapsed time value like a duration. This class has properties that represent the exact time down to millisecond precision within a calendar day. Note that this class has no reference to the calendar date or region.
